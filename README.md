# Основные сведения
В проекте есть немного несоответствий с макетом из-за неточности маржинов и разных неточностей картинок из макета (некоторые картинки там отображаются не полностью, а скачиваются они в полном размере и редактировать их под макет я не стал). Но в основном все подходит.
